
# SimpleStore

A small ecommerce app to experiment with "print on demand" services

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file. 

Create a .env file in the app directory and add these variables.


`NODE_ENV`

`MONGO_DB_DATABASE_URL`

`AUTH0_SECRET`

`APP_BASE_URL`

`AUTH0_DOMAIN`

`AUTH0_CLIENT_ID`

`AUTH0_CLIENT_SECRET`

`AUTH0_SCOPE`

`STRIPE_SECRET_KEY`

`NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`

`RESEND_API_KEY`

## Installation

Install SimpleStore with npm

```bash
  npm install
```
    
## Running Tests

To run tests, run the following command

```bash
  npm run test
```


## Tech Stack

**Tech:** React (Next.js), Vercel, Auth0, Stripe, MongoDB (Atlas), Google Cloud (Bucket)

**Styling:** TailwindCSS (DaisyUI)
